## 1) 정수형(printf/scanf 공통)

| C/C++ 타입             | printf |  scanf | 비고                  |
| -------------------- | -----: | -----: | ------------------- |
| `int`                |   `%d` |   `%d` | 부호 있는 10진           |
| `unsigned int`       |   `%u` |   `%u` | 부호 없는 10진           |
| `short`              |  `%hd` |  `%hd` | `h`                 |
| `unsigned short`     |  `%hu` |  `%hu` |                     |
| `long`               |  `%ld` |  `%ld` | `l`                 |
| `unsigned long`      |  `%lu` |  `%lu` |                     |
| `long long`          | `%lld` | `%lld` | `ll`                |
| `unsigned long long` | `%llu` | `%llu` |                     |
| `char`(정수로 출력)       | `%hhd` | `%hhd` | `signed char`일 때 안전 |
| `unsigned char`(정수)  | `%hhu` | `%hhu` |                     |

### 16진/8진/기타

| 목적       | printf | scanf | 비고           |
| -------- | -----: | ----: | ------------ |
| 16진(소문자) |   `%x` |  `%x` | unsigned로 취급 |
| 16진(대문자) |   `%X` |  `%X` |              |
| 8진       |   `%o` |  `%o` |              |

## 2) 크기 기반 타입(매우 중요)

| 타입          | printf | scanf | 비고             |
| ----------- | -----: | ----: | -------------- |
| `size_t`    |  `%zu` | `%zu` | 배열 크기/문자열 길이 등 |
| `ptrdiff_t` |  `%td` | `%td` | 포인터 차이         |
| `intmax_t`  |  `%jd` | `%jd` | 최대 폭 정수        |
| `uintmax_t` |  `%ju` | `%ju` |                |

> 코테에서 `size_t`는 실수 포인트가 많아서 **`%zu`**는 꼭 기억하시는 걸 권장합니다.

---

## 3) 실수형(printf/scanf 차이 있음)

| 타입            | printf | scanf | 비고                         |
| ------------- | -----: | ----: | -------------------------- |
| `float`       |   `%f` |  `%f` | **scanf에서 `%f`는 float***   |
| `double`      |   `%f` | `%lf` | **scanf에서 double*는 `%lf`** |
| `long double` |  `%Lf` | `%Lf` |                            |

* `printf`는 `float`도 가변 인자에서 `double`로 승격되므로 출력은 `%f`로 합니다.
* 하지만 `scanf`는 **포인터 타입 그대로** 받기 때문에 `double`은 `%lf`가 필요합니다.

## 4) 문자/문자열/포인터

| 목적          | printf |   scanf | 비고                        |
| ----------- | -----: | ------: | ------------------------- |
| 문자 1개       |   `%c` |    `%c` | scanf의 `%c`는 공백/개행도 읽음    |
| 공백 무시 문자 입력 |      — | `" %c"` | 앞 공백 1칸이 핵심               |
| 문자열(공백 전까지) |   `%s` |    `%s` | scanf는 버퍼 오버런 주의(폭 지정 권장) |
| 포인터 주소      |   `%p` |    `%p` | `void*`로 캐스팅하는 습관 권장      |
| 개행 포함 한 줄   |   (직접) | `fgets` | scanf로 “한 줄”은 까다로움        |

## 5) 자주 쓰는 옵션(printf)

| 기능      | 예시      | 의미          |
| ------- | ------- | ----------- |
| 폭 지정    | `%5d`   | 최소 5칸       |
| 0 채우기   | `%05d`  | 5칸, 빈칸은 0   |
| 정밀도(실수) | `%.3f`  | 소수점 3자리     |
| 폭+정밀도   | `%8.2f` | 폭 8, 소수 2   |
| 왼쪽 정렬   | `%-5d`  | 폭 5에서 왼쪽 정렬 |
