### 메모

- 진법 변환의 기본 원리인 **N을 B로 나눈 나머지가 현재 자릿수**를 정확히 구현한 풀이
    
- 각 나머지를 아래와 같은 규칙으로 매핑하여 문제 조건을 충족함

    - `0 ~ 9` → `'0' ~ '9'`
      
    - `10 ~ 35` → `'A' ~ 'Z'`  
        
- 계산된 자릿값을 문자열의 **맨 앞에 삽입**하여 상위 자릿수부터 올바른 순서로 결과를 구성함
    
- `Next < B`인 경우를 별도로 처리하여 마지막 자릿수를 명확히 종료 조건으로 사용함

### GPT 피드백

- 알고리즘 자체는 **진법 변환의 정석적인 사고 흐름**과 정확히 일치함
    
- 다만 구현 관점에서 다음과 같은 개선 여지가 있음
    
    - `std::string::insert(S.begin(), ...)`는 매번 문자열 전체를 이동시키므로 시간복잡도가 O(L²)이 됨
        
    - 일반적으로는 하단의 방식이 더 정석적임
        
        - 뒤에 `push_back`으로 추가한 뒤 `reverse`
            
        - 또는 스택/임시 버퍼 사용  
            
- `while (true)` + `break` 구조는 동작에는 문제가 없음
  
- 그러나 조건 기반 루프(`while (Next > 0)`)가 가독성 측면에서 더 명확할 수 있음

### 정석 코드 제안
#### 정석 코드
```cpp
#include <iostream>
#include <string>
#include <algorithm>

int main()
{
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    int N, B;
    std::cin >> N >> B;

    std::string result;

    while (N > 0)
    {
        int remain = N % B;
        N /= B;

        if (remain < 10)
            result.push_back(remain + '0');
        else
            result.push_back(remain - 10 + 'A');
    }

    std::reverse(result.begin(), result.end());
    std::cout << result;
    return 0;
}
```
