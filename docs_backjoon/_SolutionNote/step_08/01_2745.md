### GPT 피드백

- 이 문제에서의 핵심은 **진법 수의 누적 정의를 이해하고 있는지 여부**

#### 진법 수의 누적 정의

- 수학적 정의: $$((A × B) + B) × B + C$$
- 이전까지 계산한 값에 진법 B를 곱하고 현재 자리값을 더하는 방식
- 진법을 곱하면 이전까지 계산한 값의 자리수를 왼쪽으로 1칸 옮기겠다는 것
- `result = result * B + digit`

---
### 정석 코드 제안
#### 정석 코드
```cpp
#include <iostream>
#include <string>

int main()
{
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    std::string S;
    int B;
    std::cin >> S >> B;

    long long result = 0;

    for (char c : S)
    {
        int digit;
        if (c >= '0' && c <= '9')
            digit = c - '0';
        else
            digit = c - 'A' + 10;

        result = result * B + digit;
    }

    std::cout << result;
    return 0;
}
```
