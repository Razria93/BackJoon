### GPT 피드백

- 개선 필요(정석)
  입력 개수 고정(N=9) → while (cin >> x) 대신 for (int i=1; i<=9; ++i) 사용

- 개선(명확성)
  변수명 num/valueIdx → index/maxIndex처럼 “1-based 인덱스” 의미가 드러나게 정리

- 참고(규칙)
  while (cin >> ...)는 “EOF까지 입력” 문제(예: 10951/11718)에서만 기본 템플릿으로 사용
