### GPT 피드백

- 그러나 카운팅하는 과정을 훨씬 간단하게 표현할 수 있었음
	- 본인 풀이: 
		1. 입력값이 들어오면 `map.find(X)`
		2. `find`로 반환받은 `iterator` 유효성 체크
		3. 유효하지 않다면 `insert`
		4. 유효하다면 `iterator`로 해당 `pair`의 `second`에 접근하여 값을 증가시킴

- 권장풀이:
	-  입력값이 들어오면 `++map[X]`
		- 해당 방식에 key 생성, 값 조회 및 증가가 포함되어 있음

### 정석 코드 제안
#### 정석 코드
```cpp
#include <iostream>
#include <unordered_map>

int main()
{
    std::ios::sync_with_stdio(0);
    std::cin.tie(0);

    int N;
    std::cin >> N;

    std::unordered_map<int, int> cnt;
    cnt.reserve(N);

    for (int i = 0; i < N; ++i)
    {
        int x;
        std::cin >> x;
        ++cnt[x];
    }

    int M;
    std::cin >> M;

    for (int i = 0; i < M; ++i)
    {
        int x;
        std::cin >> x;

        auto it = cnt.find(x);
        if (it != cnt.end()) std::cout << it->second;
        else std::cout << 0;

        if (i + 1 < M) std::cout << ' ';
    }

    std::cout << '\n';
    return 0;
}
```
