### 정석 코드 제안
#### 정석 코드 (reverse 기반)

```cpp
#include <iostream>
#include <string>
#include <algorithm>

int main()
{
    std::string s;
    std::cin >> s;

    std::string r = s;
    std::reverse(r.begin(), r.end());

    std::cout << (s == r);
    return 0;
}
```

#### 특징
- 구현이 가장 직관적
- 문자열 문제 입문 단계에서 적합

#### 정석 코드 (양 끝 포인터 기반)

```cpp
#include <iostream>
#include <string>

int main()
{
    std::string s;
    std::cin >> s;

    int left = 0;
    int right = s.size() - 1;

    while (left < right)
    {
        if (s[left] != s[right])
        {
            std::cout << 0;
            return 0;
        }
        ++left;
        --right;
    }

    std::cout << 1;
    return 0;
}

```

#### 특징
- 추가 문자열 생성 없음
- 비교 횟수 최소화
- 코딩 테스트에서 가장 권장되는 방식
