### 메모

- 문제에서 요구하는 체스 말의 **정확한 기준 개수**를 알고 있어야 함  
    (킹 1, 퀸 1, 룩 2, 비숍 2, 나이트 2, 폰 8)
- 입력값은 현재 보유 개수이므로 **기준값 − 입력값**을 그대로 출력
- 반복문이나 배열 없이도 충분히 해결 가능

### GPT 피드백

- 문제 요구를 정확히 해석한 **정직한 O(1) 풀이**
- 불필요한 자료구조나 반복문 없이 바로 산술 연산으로 해결한 점이 바람직함
- 가독성을 위해 기준 개수를 배열로 두는 방식도 가능하지만, 이 문제 규모에서는 현재 방식이 더 명확함
- 입출력 최적화 설정은 필수는 아니지만 습관적으로 유지해도 무방함
