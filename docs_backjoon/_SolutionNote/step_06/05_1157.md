### 정석 코드 제안
#### 정석 코드
```cpp
#include <iostream>
#include <string>
#include <cctype>

int main()
{
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    std::string S;
    std::cin >> S;

    int freq[26] = {0};

    for (char c : S)
    {
        c = std::toupper(c); // 문자를 대문자료 변환
        freq[c - 'A']++;
    }

    int maxCount = 0;
    char answer = '?';
    bool duplicated = false;

    for (int i = 0; i < 26; ++i)
    {
        if (freq[i] > maxCount)
        {
            maxCount = freq[i];
            answer = char('A' + i);
            duplicated = false;
        }
        else if (freq[i] == maxCount && freq[i] != 0)
        {
            duplicated = true;
        }
    }

    std::cout << (duplicated ? '?' : answer);
    return 0;
}
```

#### 참고
- `#include <cctype>`
- `std::toupper`: 문자를 대문자로 정규화하는 함수
- `std::tolower`: 문자를 소문자로 정규화하는 함수
- `char`을 인수로 넣고 싶은 경우 반드시 `static_cast<unsigned char>(..)`로 형변환
