- - 문제 조건(정수 5개, 평균은 정수로 출력)에 맞게 **정수 나눗셈**(`Sum/5`)을 사용한 것이 정확함
    
- 중앙값은 **정렬 후 인덱스 2**가 맞음(0-based)

### 정석 코드 제안
#### 정석 코드
```cpp
#include <iostream>
#include <algorithm>

int main() {
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    int a[5];
    int sum = 0;

    for (int i = 0; i < 5; ++i) 
    {
        std::cin >> a[i];
        sum += a[i];
    }

    std::sort(a, a + 5);

    std::cout << sum / 5 << '\n';
    std::cout << a[2];

    return 0;
}
```
