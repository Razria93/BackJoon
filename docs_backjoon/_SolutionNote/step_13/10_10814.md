### 메모

- `stable_sort` 첫 사용이며, 해당 API 자체를 모르고 있었으므로 결과는 FAIL

#### `stable_sort`

- `std::sort` : **불안정 정렬** (동등한 원소들의 상대 순서가 유지된다는 보장이 없음)
    
- `std::stable_sort` : **안정 정렬** (동등한 원소들의 상대 순서를 _입력 순서대로 유지_)

#### 동등한 원소란?

- 정렬 기준(Comparator)에서 두 원소 `a`, `b`에 대해 다음과 같을 경우 `a`와 `b`는 정렬 기준에서 **동등(equivalent)** 한 원소임
  
```cpp
comp(a, b) == false && comp(b, a) == false
```

### 정석 코드 제안
#### 정석 코드
```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

int main()
{
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    int n;
    std::cin >> n;

    std::vector<std::pair<int, std::string>> a;
    a.reserve(n);

    for (int i = 0; i < n; ++i)
    {
        int age;
        std::string name;
        std::cin >> age >> name;

        a.push_back({ age, name });
    }

    std::stable_sort(a.begin(), a.end(),
        [](const auto& lhs, const auto& rhs)
        {
            return lhs.first < rhs.first;
        });

    for (const auto& p : a)
    {
        std::cout << p.first << ' ' << p.second << '\n';
    }

    return 0;
}
```
