### GPT 피드백

- 현재 풀이로 정답 처리됨
    
- `A.reserve(N)`을 추가하면 `push_back` 중 재할당을 줄일 수 있음(필수 아님)
    
- 출력은 `'\n'` 사용이 적절함(불필요한 flush 방지)

---
### 정석 코드 제안
#### 정석 코드
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int main() 
{
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    int N;
    std::cin >> N;

    std::vector<int> A;
    A.reserve(N);

    for (int i = 0; i < N; ++i) 
    {
        int x;
        std::cin >> x;
        A.push_back(x);
    }

    std::sort(A.begin(), A.end());

    for (int x : A) 
    {
        std::cout << x << '\n';
    }

    return 0;
}
```
