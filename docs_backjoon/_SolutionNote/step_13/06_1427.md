### GPT 피드백

- 로직은 문제 요구사항(자리수 내림차순 정렬 후 그대로 출력)과 일치함
    
- `S.size()` / `A.size()`는 `size_t`라서 현재처럼 `int` 인덱스와 섞으면 경고가 날 수 있음(동작은 보통 문제 없음). 인덱스 타입을 `size_t`로 맞추는 편이 깔끔함
    
- 이 문제는 자리수가 최대 10개 수준이라 정렬/벡터 없이도 간단히 처리 가능
    
    - 방법1: 문자열을 그대로 `std::sort(S.begin(), S.end(), std::greater<char>())` 후 출력
        
    - 방법2: 카운팅(0~9)으로 더 단순하게 구현 가능

### 정석 코드 제안
#### 정석 코드
```cpp
#include <iostream>
#include <string>
#include <algorithm>
#include <functional>

int main()
{
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    std::string s;
    std::cin >> s;

    std::sort(s.begin(), s.end(), std::greater<char>());

    std::cout << s;

    return 0;
}
```
