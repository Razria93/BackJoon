### 메모

- 삼각형을 판단하는 순서와 조건을 잘못 작성함

- 분류 기준
    
    - **Equilateral**: `A == B == C`
    - **Scalene**: `A != B && B != C && C != A`
    - **Isosceles**: 그 외 (두 각만 같은 경우)
  
---

### 정석 코드 제안
#### 정석 코드
```cpp
#include <iostream>

int main()
{
    std::ios::sync_with_stdio(0);
    std::cin.tie(0);

    int A, B, C;
    std::cin >> A >> B >> C;

    if (A + B + C != 180)
    {
        std::cout << "Error";
    }
    else if (A == B && B == C)
    {
        std::cout << "Equilateral";
    }
    else if (A != B && B != C && C != A)
    {
        std::cout << "Scalene";
    }
    else
    {
        std::cout << "Isosceles";
    }

    return 0;
}
```
