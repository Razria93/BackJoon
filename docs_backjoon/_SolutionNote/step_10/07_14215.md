### 메모

- 삼각형의 변을 만들 수 있는 막대를 `줄일 수만 있다` 가 키워드
  
- 최초로 주어진 막대로 온전한 삼각형을 만들 수 있다면 해당 삼각형이 최대 크기임
  
- 삼각형을 구성할 수 있는 최소 기준은 `최대 변의 길이 < 나머지 두 변의 길이 합`을 만족하는 것
  
- 따라서 세 막대가 삼각형을 구성할 때 필요한 기준에 맞지 않는다면 `최대 변의 길이 = 나머지 두변의 길이 합 - 1`를 기준으로 삼각형을 구성할 수 있는 최대의 변 길이로 수정함

### GPT 피드백

- 문제의 핵심을 **삼각형을 만들 수 있도록 가장 긴 변을 줄인다**로 정확히 파악함
    
- 불필요한 정렬 없이 `max` 기반 분기 처리 → 효율적
    
- 실제 수학적 조건을 코드로 잘 치환함
    
- 이 문제는 구현 문제가 아니라 **삼각형 부등식 이해 여부를 확인하는 문제**

### 정석 코드 제안
#### 정석 코드
```cpp
#include <iostream>
#include <algorithm>

int main()
{
    std::ios::sync_with_stdio(0);
    std::cin.tie(0);

    int A, B, C;
    std::cin >> A >> B >> C;


    int Max = std::max({A, B, C});
    int Sum = A + B + C;

    if (Max >= Sum - Max)
    {
        Max = (Sum - Max) - 1;
    }
	
    std::cout << (Sum - std::max({A, B, C}) + Max); // Sum - OldMax + NewMax
    return 0;
}
```
