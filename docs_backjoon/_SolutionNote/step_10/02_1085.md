### 메모

- 탈출까지의 최소 거리는 다음 네 값 중 최소
    
    - `X` (왼쪽 변)
    - `W - X` (오른쪽 변)
    - `Y` (아래 변)
    - `H - Y` (위쪽 변)
        
- 이를 가로 / 세로로 나누어 `min`을 두 번 적용하면 충분
    
- 조건 분기 없이 **수식으로 바로 해결 가능**
  
---
### GPT 피드백

- 문제를 거리 문제로 정확히 해석함
    
- 불필요한 분기나 반복문 없이 **최소 연산만 사용**
    
- `std::min`을 단계적으로 적용한 구조가 가독성 좋음
    
- 이 문제는 좌표·기하 문제의 **기본 패턴 확인용 문제**

### 정석 코드 제안
#### 정석 코드
```cpp
#include <iostream>
#include <algorithm>

int main()
{
    std::ios::sync_with_stdio(0);
    std::cin.tie(0);

    size_t X, Y, W, H;
    std::cin >> X >> Y >> W >> H;

    size_t dx = std::min(X, W - X);
    size_t dy = std::min(Y, H - Y);

    std::cout << std::min(dx, dy);
    return 0;
}
```
