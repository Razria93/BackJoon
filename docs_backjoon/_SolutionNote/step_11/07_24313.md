### 메모

- 모든 `n ≥ n0` 에 대하여 $a_1 * ​n + a_0 ​≤ c * n$ 를 만족해야함

- $d = a_1​ − c ⇒ d * n + a_0 ​≤ 0$ 으로 정리할 수 있으며, 
	- $c$는 $1 ≤ c ≤ 100$ 의 양의 정수
	- $a_0$ 와 $a_1$ 은 $0 ≤ |ai| ≤ 100$ 의 정수

- 이 때, `n`은 최악의 값 $n_0$ 부터 양의 방향으로 발산
	- $n_0$는 $1 ≤ n_0 ≤ 100$ 의 양의 정수

- 해당 문제는 $d$ 를 기준으로 $d$ 가 양수일 때 / 0 일 때 / 음수일 때의 Case로 나누어 각 Case 마다 디테일한 조건을 도출해야 함

#include <iostream>

int main()
{
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    long long a1, a0, c, n0;
    std::cin >> a1 >> a0 >> c >> n0;

    long long d = a1 - c;

    bool ok = false;

    if (d > 0)
    {
        ok = false;
    }
    else if (d == 0)
    {
        ok = (a0 <= 0);
    }
    else
    {
        // d < 0: worst case at n = n0
        // d*n0 + a0 <= 0 ?
        __int128 lhs = (__int128) d * n0 + a0;
        ok = (lhs <= 0);
    }

    std::cout << (ok ? 1 : 0);
    return 0;
}
